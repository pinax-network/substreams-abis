# WOOFi

Cross-chain DEX powered by WOO Network's market-making liquidity, offering low-fee swaps and cross-chain transactions.

## Key Events

- **`WooRouterSwap`** — Emitted on every swap through the WOOFi router
- **`WooPoolChanged`** — Emitted when the underlying WooPool address is updated

## Overview

### WooRouterV2
The main swap router that routes trades through WOO Network's synthetic proactive market making (sPMM) pools. Provides institutional-grade liquidity with tight spreads.
