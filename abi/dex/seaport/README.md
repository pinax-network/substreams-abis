# Seaport

OpenSea's decentralized marketplace protocol for trading NFTs and other digital assets with flexible order types.

## Contracts

| Contract | Address | Chain | Explorer |
|----------|---------|-------|----------|
| Seaport | Various deployments | Multi-chain | — |

## Key Events

- **`OrderFulfilled`** — Emitted when an order is successfully filled
- **`OrderCancelled`** — Emitted when an order is cancelled
- **`OrderValidated`** — Emitted when an order is validated on-chain
- **`OrdersMatched`** — Emitted when multiple orders are matched together
- **`CounterIncremented`** — Emitted when a user increments their counter to cancel all open orders

## Overview

### Seaport
The core marketplace contract supporting a wide variety of order types including basic listings, English auctions, Dutch auctions, and collection offers. Supports ERC721, ERC1155, ERC20, and native ETH in a single unified protocol.
