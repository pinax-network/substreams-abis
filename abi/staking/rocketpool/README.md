# Rocket Pool

Decentralized Ethereum staking protocol allowing permissionless node operation with only 8 ETH, issuing rETH as the liquid staking token.

## Contracts

| Contract | Address | Chain | Explorer |
|----------|---------|-------|----------|
| rETH | `0xae78736Cd615f374D3085123A210448E74Fc6393` | Ethereum | [Etherscan](https://etherscan.io/address/0xae78736Cd615f374D3085123A210448E74Fc6393) |

## Key Events

- **`TokensMinted`** — Emitted when rETH is minted in exchange for ETH deposits
- **`TokensBurned`** — Emitted when rETH is burned to redeem ETH
- **`EtherDeposited`** — Emitted when ETH is deposited into the contract
- **`Transfer`** — Standard ERC-20 transfer event

## Overview

### rETH
The Rocket Pool liquid staking token. Uses an exchange rate model where rETH appreciates against ETH as staking rewards are earned by the network's decentralized node operators.
